# Math - Euler Squares, aka Double Latin Squares, aka Graeco-Latin Squares

## Euler Squares - Numberphile
* https://www.youtube.com/watch?v=qu04xLNrk94

## NOTE:
* 2x2 cannot be done.
* 6x6 cannot be done.
